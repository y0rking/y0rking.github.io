使用hexo+github搭建自己的免费博客

发表于 2018-07-09 | 更新于 2019-06-04

准备工作

  • 安装 node
  • 安装 gitbash
  • 使用npm安装hexo
    本人环境是window10

一、安装node

从node官网下载最新的正式版，直接 下一步、下一步直到安装完成 。完成 之后使用win+R打开命令窗口，

若如上所示，显示了版本号，则表示安装成功。

二、安装gitbash

从gitbash官网下载最新版，直接安装直到完成后，打开win+R窗口：

三、安装 hexo

npm安装淘宝源，此操作是在gitbash里进行，后面所有的执行命令的操作都在gitbash里，避免windows下不必要的麻烦

$ npm install -g cnpm --registry=https://registry.npm.taobao.org

之后 则可以使用cnpm命令安装模块了， 速度 杠杠的。
安装hexo
$ cnpm install -g hexo-cli
选择一个想要创建博客的目录，如myblog
$ hexo init myblog
$ cd myblog
$ npm install

创建github工程

注册github帐号并登录，创建一个和自己userame一致的项目，如username为testname,则项目名称为testname.github.io。创建成功后，访问testname.github.io地址，若成功打开空白则表示配置成功。

OK， github.io的访问地址有了，我们只需要把本地hexo生成的静态页面部署到此项目就可以了。

创建 hexo 项目

从gitbash中进入刚刚创建的myblog 目录，执行
$ hexo g # 生成静态页面
我们可以看到 myblog目录下多一个public目录，public目录就是我们需要部署到github的内容。OK，现在我们看看，生成的页面是什么样子的？
执行命令：
npm install hexo-server –save #hexo 3.0 需要独立安装server模块
$ hexo s #启动服务
然后访问http://localhost:4000/(http://localhost:4000)，即可预览页面郊果。

上传至github

还差最后一步，上传至github,我们就大功告成了。
首先配置好ssh key,在gitbash里执行
$ ssh-keygen -t rsa -C “你的邮箱地址”
找到生成的key 并添加至 github的 ssh keys中。
配置项目中github的仓库地址，注意：此处不能使用https访问地址。
编辑项目根目录下的 _config.yml 文件。
